## 執筆者向け情報

### 必要なツール

- Nix
  - Nix commands及びFlakes有効化済み

### コマンド

#### プレビュー

[localhost:8000](http://localhost:8000)でzenn-cliが起動します。

```bash
nix run
```

#### フォーマット

以下の形式のファイルをフォーマットします。

- `*.nix`
- `*.md`

```bash
nix fmt
```
